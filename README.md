# Лабораторная работа №1. Работа с Git и GitHub

## Описание проекта
Консольное приложение на языке C++, предназначенное для расчета геометрических параметров треугольника. Проект выполнен в рамках изучения систем контроля версий (Git) и хостинга репозиториев (GitHub).

**Вариант задания:** 1.b (Геометрические фигуры: Треугольник)

## Функциональные возможности
Программа запрашивает у пользователя длины трех сторон и выполняет следующие операции:
1. **Валидация данных:** Проверка возможности построения треугольника по неравенству сторон ($a + b > c$).
2. **Расчет периметра ($P$):** Нахождение суммы длин всех сторон.
3. **Расчет площади ($S$):** Использование формулы Герона через полупериметр ($p$).
4. **Анализ типа фигуры:** Определение, является ли треугольник равнобедренным.

## Листинг программы (treg.cpp)
```cpp
#include <iostream>
#include <cmath>

int main() {
    double a, b, c;
    std::cout << "Введите три стороны треугольника: ";
    std::cin >> a >> b >> c;

    if (a + b <= c || a + c <= b || b + c <= a) {
        std::cout << "Треугольник не существует" << std::endl;
        return 1; 
    }

    double P = a + b + c;
    double p = P / 2;
    double S = sqrt(p * (p - a) * (p - b) * (p - c));

    std::cout << "Периметр: " << P << std::endl;
    std::cout << "Площадь: " << S << std::endl;

    if (a == b || b == c || a == c) {
        std::cout << "Треугольник равнобедренный." << std::endl;
    } else {
        std::cout << "Треугольник не равнобедренный." << std::endl;
    }
    return 0;
}
